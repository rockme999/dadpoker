1. 시작하기

1.1 앱 소개

DAD Poker는 포커 플레이어를 위한 종합 학습 및 관리 앱입니다. 뱅크롤 관리, 학습 자료, 포커 계산 도구, 커뮤니티 기능을 제공하여 포커 실력을 향상시키고 게임을 체계적으로 관리할 수 있습니다.

그림 1-1: DAD Poker 메인 화면

1.2 회원가입 및 로그인

앱을 처음 실행하면 로그인 화면이 표시됩니다. 다음 방법으로 계정을 생성할 수 있습니다:

회원가입 방법

  1. 1 이메일 회원가입: 이메일과 비밀번호를 입력하여 계정 생성
  2. 2 Google 로그인: Google 계정으로 간편 로그인
  3. 3 Apple 로그인: iOS 기기에서 Apple ID로 로그인
그림 1-2: 로그인 화면
💡 팁: 회원가입 시 600 DAD머니가 자동으로 지급됩니다. 이를 활용하여 학습 자료를 구매하거나 다양한 기능을 이용할 수 있습니다.

1.3 프로필 완성

첫 로그인 시 닉네임을 설정해야 합니다. 닉네임은 커뮤니티에서 표시되며, 중복 확인이 필요합니다.

그림 1-3: 프로필 설정 화면

2. 대시보드

대시보드는 앱의 홈 화면으로, 모든 주요 기능의 요약 정보를 한눈에 볼 수 있습니다.

그림 2-1: 대시보드 화면

2.1 주요 섹션

뱅크롤 현황

현재 보유 중인 뱅크롤 목록과 총 자금을 확인할 수 있습니다. 각 뱅크롤의 현재 잔액과 통화 정보가 표시됩니다.

그림 2-2: 뱅크롤 현황

최근 활동

최근에 사용한 도구, 학습한 자료, 커뮤니티 활동 등을 빠르게 확인하고 다시 접근할 수 있습니다.

그림 2-3: 최근 활동

빠른 접근

자주 사용하는 도구나 학습 자료에 빠르게 접근할 수 있는 바로가기 링크가 제공됩니다.

3. 뱅크롤 관리

뱅크롤 화면에서는 포커 게임에 사용하는 자금을 체계적으로 관리할 수 있습니다.

그림 3-1: 뱅크롤 관리 화면

3.1 뱅크롤 추가

  1. 1 뱅크롤 화면에서 + 버튼 클릭
  2. 2 뱅크롤 이름 입력 (예: "카지노 A", "온라인 B")
  3. 3 초기 자금 입력
  4. 4 통화 선택 (원화, 달러, 유로 등)
  5. 5 저장 버튼 클릭
그림 3-2: 뱅크롤 추가 화면

3.2 거래 기록

각 뱅크롤의 입금, 출금, 게임 결과 등을 기록하여 자금 흐름을 추적할 수 있습니다.

그림 3-3: 거래 기록 화면

거래 유형

유형 설명
입금 뱅크롤에 자금을 추가하는 경우
출금 뱅크롤에서 자금을 인출하는 경우
게임 결과 포커 게임에서 얻은 수익 또는 손실

3.3 통계 및 분석

뱅크롤별 수익률, 총 손익, 기간별 성과 등을 그래프와 통계로 확인할 수 있습니다.

그림 3-4: 뱅크롤 통계 화면

4. 스터디

스터디 화면에서는 포커 학습 자료를 탐색하고, 구매하며, 학습할 수 있습니다.

그림 4-1: 스터디 화면

4.1 패키지 탐색

스터디 화면은 세 개의 탭으로 구성되어 있습니다:

탭 구성

  • 전체: 모든 학습 패키지 (시스템 + 사용자 생성)
  • 공식: DAD Poker에서 제공하는 공식 학습 자료
  • 사용자: 다른 사용자들이 만든 학습 자료
그림 4-2: 스터디 탭 구성

4.2 패키지 구매

  1. 1 원하는 패키지 선택
  2. 2 패키지 상세 정보 확인 (제목, 설명, 가격, 목차)
  3. 3 구매하기 버튼 클릭
  4. 4 DAD머니가 차감되며 패키지 구매 완료
  5. 5 자료 보기 버튼으로 학습 시작
그림 4-3: 패키지 상세 화면
⚠️ 주의: 패키지 구매 후에는 환불이 불가능합니다. 구매 전에 패키지 정보를 충분히 확인하세요.

4.3 학습하기

구매한 패키지는 오프라인에서도 학습할 수 있습니다. 자료는 순서대로 학습하며, 각 자료에 댓글을 남기거나 추천할 수 있습니다.

그림 4-4: 학습 자료 뷰어

자료 유형

  • 텍스트: 일반 텍스트 기반 학습 내용
  • 이미지: 이미지와 설명이 포함된 자료
  • 카드 위젯: 커뮤니티 스타일의 카드 형태 자료
  • 차트 위젯: 데이터 시각화 자료

4.4 나만의 패키지 만들기

자신만의 학습 자료를 만들어 판매할 수 있습니다. 판매 수익의 80%는 저작자에게, 20%는 플랫폼 수수료로 지급됩니다.

  1. 1 스터디 화면에서 + 버튼 클릭
  2. 2 패키지 제목, 설명, 가격 설정
  3. 3 메인 이미지 설정 (선택사항)
  4. 4 학습 자료 추가 (텍스트, 이미지, 위젯)
  5. 5 발행 버튼으로 패키지 등록
그림 4-5: 패키지 편집 화면

5. 도구

도구 화면에서는 포커 게임에 유용한 다양한 계산기를 제공합니다.

그림 5-1: 도구 화면

5.1 확률 계산기

핸드 승률과 확률을 계산합니다. 보드 카드와 상대방의 가능한 핸드를 입력하면 승률을 계산해줍니다.

그림 5-2: 확률 계산기

5.2 ICM 계산기

토너먼트에서 Independent Chip Model(ICM)을 사용하여 칩의 실제 가치를 계산합니다.

그림 5-3: ICM 계산기

5.3 GTO 차트

Game Theory Optimal(GTO) 전략 차트를 생성하여 최적의 플레이를 분석합니다.

그림 5-4: GTO 차트

5.4 아웃츠 계산

아웃 카드와 확률을 분석하여 다음 카드로 승리할 확률을 계산합니다.

그림 5-5: 아웃츠 계산기

5.5 올인 계산기

멀티 플레이어 상황에서 올인 시 팟 분배를 계산합니다.

그림 5-6: 올인 계산기

5.6 ROI 계산기

투자 대비 수익률(Return on Investment)을 계산하여 게임 성과를 분석합니다.

그림 5-7: ROI 계산기

6. 커뮤니티

커뮤니티 화면에서는 다른 포커 플레이어들과 소통하고 정보를 공유할 수 있습니다.

그림 6-1: 커뮤니티 화면

6.1 게시글 작성

  1. 1 커뮤니티 화면에서 + 버튼 클릭
  2. 2 제목과 내용 입력
  3. 3 이미지 첨부 (선택사항)
  4. 4 작성 버튼 클릭
그림 6-2: 게시글 작성 화면

6.2 댓글 및 추천

게시글에 댓글을 남기거나 추천할 수 있습니다. 유용한 정보를 공유하는 게시글은 추천해주세요.

그림 6-3: 게시글 상세 화면

6.3 메시지

다른 사용자에게 개인 메시지를 보낼 수 있습니다. 메시지는 실시간으로 전달됩니다.

그림 6-4: 메시지 화면

7. 설정

설정 화면에서는 앱의 다양한 옵션을 관리할 수 있습니다.

그림 7-1: 설정 화면

7.1 테마 설정

구독 사용자는 앱의 테마 모드(라이트/다크/시스템)와 주제 색상을 변경할 수 있습니다.

그림 7-2: 테마 설정

7.2 계정 관리

  • 프로필 보기: 닉네임, 등급, DAD머니 확인
  • 비밀번호 변경: 이메일 로그인 사용자만 가능
  • 로그아웃: 현재 계정에서 로그아웃
  • 계정 삭제: 계정 및 모든 데이터 삭제 (복구 불가)
⚠️ 경고: 계정 삭제는 되돌릴 수 없습니다. 삭제 전에 모든 데이터가 영구적으로 삭제됨을 확인하세요.

7.3 구독

구독을 통해 모든 기능을 제한 없이 사용할 수 있습니다. 구독 시 광고가 제거되고 모든 프리미엄 기능이 활성화됩니다.

그림 7-3: 구독 화면

8. 자주 묻는 질문

Q1. DAD머니는 어떻게 충전하나요?

A: 현재는 회원가입 시 지급되는 600 DAD머니를 사용할 수 있습니다. 추후 인앱 결제를 통해 충전할 수 있는 기능이 추가될 예정입니다.

Q2. 구매한 패키지는 오프라인에서도 볼 수 있나요?

A: 네, 구매한 패키지는 로컬에 저장되어 오프라인에서도 학습할 수 있습니다. 다만, 최신 업데이트를 받으려면 인터넷 연결이 필요합니다.

Q3. 패키지를 구매했는데 환불이 가능한가요?

A: 현재는 패키지 구매 후 환불이 불가능합니다. 구매 전에 패키지 정보를 충분히 확인하시기 바랍니다.

Q4. 내가 만든 패키지의 수익은 언제 지급되나요?

A: 다른 사용자가 패키지를 구매하면 즉시 DAD머니로 수익의 80%가 지급됩니다.

Q5. 구독을 취소하면 어떻게 되나요?

A: 구독 취소 시 현재 구독 기간이 끝날 때까지는 모든 기능을 사용할 수 있습니다. 기간 종료 후에는 무료 기능만 사용 가능합니다.

Q6. 데이터는 어디에 저장되나요?

A: 모든 데이터는 Supabase 클라우드에 저장되며, 구매한 학습 자료는 로컬에도 캐싱되어 오프라인 접근이 가능합니다.
