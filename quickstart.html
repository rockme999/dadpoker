🚀 DAD Poker 퀵스타트 가이드

5분 안에 앱의 주요 기능을 익혀보세요!

1. 로그인하기

앱을 처음 실행하면 로그인 화면이 나타납니다.

  1. 구글 또는 애플 계정으로 간편 로그인
  2. 프로필 설정 (닉네임, 프로필 사진)
그림 1: 로그인 화면

2. 대시보드 둘러보기

로그인 후 메인 대시보드에서 모든 주요 기능에 접근할 수 있습니다.

  • 뱅크롤: 자금 관리 및 통계
  • 스터디: 학습 패키지 탐색
  • 도구: 확률 계산기, ICM 등
  • 커뮤니티: 게시판 및 소통
그림 2: 메인 대시보드

3. 뱅크롤 기록하기

게임 결과를 기록하여 수익을 추적할 수 있습니다.

  1. 뱅크롤 탭 선택
  2. 우측 상단 + 버튼 클릭
  3. 게임 정보 입력 (금액, 날짜, 메모)
  4. 저장 클릭
그림 3: 뱅크롤 화면

4. 학습 자료 탐색하기

스터디 탭에서 포커 학습 패키지를 둘러보세요.

  • 공식 패키지: 검증된 전문 콘텐츠
  • 사용자 제작: 커뮤니티 공유 자료
  • DAD머니로 구매하여 오프라인에서도 학습 가능
그림 4: 스터디 화면

5. 도구 활용하기

포커 계산에 필요한 다양한 도구를 활용하세요.

  • 확률 계산기
  • 올인 EV 계산기
  • ICM 계산기
  • GTO 차트
그림 5: 도구 화면

6. 커뮤니티 참여하기

다른 플레이어들과 소통하고 경험을 공유하세요.

  • 게시글 작성 및 댓글
  • 다른 사용자와 메시지
  • 핸드 히스토리 공유
그림 6: 커뮤니티 화면

7. 설정 확인하기

설정에서 앱을 개인화하고 계정을 관리하세요.

  • 테마 변경 (다크/라이트 모드)
  • 알림 설정
  • 구독 관리
  • 계정 정보
그림 7: 설정 화면

💡 빠른 팁

  • ✅ DAD머니 관리: 학습 자료 구매에 신중하게 사용하세요
  • ✅ 오프라인 학습: 구매한 패키지는 오프라인에서도 볼 수 있습니다
  • ✅ 뱅크롤 기록: 게임 후 즉시 기록하면 정확한 분석이 가능합니다
  • ✅ 커뮤니티 활용: 다른 플레이어의 경험을 공유하고 배우세요
